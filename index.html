Asynchronous Programming

  • Get around functions blocking functions that prevent any code from being executed while we wait for our original functions to complete
  • Callbacks were designed for such purpose. One could pass callbacks to what would otherwise be blocking code in JS.
  • This is especially important when one has a long running operations, such as reading a file.

Callbacks

Here is an example: 


							let fs = require('fs')

							function printFileContents(error, fileContents){
    							if (error) {
        							console.log('ERROR!');
    							} else {
        							console.log(fileContents);
    							}
							}

							fs.readFile('./hello.txt', 'utf-8', printFileContents);
							console.log('Hello!');
  • The above function would read the helo.txt file.
  • The code would assign the contents of the file to the variable fileContents.
  • The code would then print fileContents.
